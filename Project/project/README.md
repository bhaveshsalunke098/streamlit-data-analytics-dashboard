# 📊 Streamlit Data Analytics Dashboard

An interactive **Data Analytics Dashboard** built using **Python** and **Streamlit** that simplifies Exploratory Data Analysis (EDA) through automated statistics, interactive visualizations, and data quality analysis.

---

## 🚀 Features

- 📂 Upload CSV datasets
- 📋 Dataset Preview
- 📊 Statistical Summary
- 🔍 Missing Value Analysis
- 📈 Interactive Data Visualizations
- 📉 Correlation Analysis
- 📦 Outlier Detection
- 📑 Data Distribution Analysis
- 🎛️ Interactive Dashboard
- ⚡ Clean & Responsive UI

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Backend |
| Streamlit | Web Framework |
| Pandas | Data Manipulation |
| NumPy | Numerical Computing |
| Matplotlib | Data Visualization |
| Plotly | Interactive Charts |

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/bhaveshsalunke098/streamlit-data-analytics-dashboard.git
```

### 2. Navigate to the Project Folder

```bash
cd streamlit-data-analytics-dashboard
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```
